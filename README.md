# 简介
本项目主要学习之用，参照示例 [incubator-weex](https://github.com/apache/incubator-weex) ，搬运代码达到自己构建weex开发环境的目的。本项目继承于[weexTest](https://github.com/xuyitao/weexTest)。

#由来
weexTest项目是单页面，在页面切换时效果不佳，故有了此项目。采用多页面跳转。
